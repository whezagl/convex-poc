## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=http://localhost:3000/api
```

## API Integration

This frontend uses TanStack Query for data fetching. API calls are defined in `src/api/` directory.

Example:
```typescript
// src/api/users.ts
import { useQuery } from '@tanstack/react-query';

export function useUsers() {
  return useQuery({
    queryKey: ['users'],
    queryFn: async () => {
      const response = await fetch('/api/users');
      return response.json();
    },
  });
}
```

## Routing

React Router is used for client-side routing. Routes are defined in `src/App.tsx`.
